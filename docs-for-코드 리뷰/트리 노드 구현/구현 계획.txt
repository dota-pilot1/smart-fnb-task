# Figma 카드 목록 기능 구현

## Context
현재 FigmaContent는 URL 1개만 저장/미리보기하는 구조.
Figma URL을 여러 개 저장하고, 카드 목록으로 보여주는 기능을 추가한다.
카드 클릭 시 새탭으로 Figma 링크 열림. 썸네일/iframe 없이 제목+설명 텍스트만 표시 (성능).

## 구현 계획

### Phase 1: 백엔드

**Step 1. FigmaLink 엔티티**
- 파일: `smart-fnb-backend/src/main/java/com/smartfnb/devspec/domain/FigmaLink.java` (신규)
- 테이블: `figma_links` (dev_spec_id FK, title, description, url, created_at, updated_at)
- 기존 DevSpecContent.java 패턴 그대로 따름

**Step 2. Repository**
- `smart-fnb-backend/src/main/java/com/smartfnb/devspec/domain/FigmaLinkRepository.java` (신규 인터페이스)
- `smart-fnb-backend/src/main/java/com/smartfnb/devspec/infrastructure/JpaFigmaLinkRepository.java` (신규 JPA 구현)
- 메서드: findByDevSpec, deleteById

**Step 3. DTO**
- `smart-fnb-backend/src/main/java/com/smartfnb/devspec/application/dto/CreateFigmaLinkRequest.java` (신규)
- `smart-fnb-backend/src/main/java/com/smartfnb/devspec/application/dto/FigmaLinkResponse.java` (신규)

**Step 4. Service에 메서드 추가**
- 파일: `smart-fnb-backend/src/main/java/com/smartfnb/devspec/application/DevSpecService.java` (수정)
- 추가: createFigmaLink, findFigmaLinks, deleteFigmaLink

**Step 5. Controller에 엔드포인트 추가**
- 파일: `smart-fnb-backend/src/main/java/com/smartfnb/devspec/presentation/DevSpecController.java` (수정)
- POST /api/devspec/{id}/figma-links → 생성
- GET /api/devspec/{id}/figma-links → 목록 조회
- DELETE /api/devspec/{id}/figma-links/{linkId} → 삭제

### Phase 2: 프론트엔드

**Step 6. 타입 추가**
- 파일: `smart-fnb-front/src/entities/devspec/model/types.ts` (수정)
- FigmaLink, CreateFigmaLinkRequest 인터페이스 추가

**Step 7. useFigmaLinks 훅**
- 파일: `smart-fnb-front/src/features/devspec/model/use-figma-links.ts` (신규)
- fetchLinks, createLink, deleteLink + 상태 관리

**Step 8. CommonCardForList 공통 컴포넌트**
- 파일: `smart-fnb-front/src/shared/ui/CommonCardForList.tsx` (신규)
- props: title, description, onClick, onDelete (optional)
- 텍스트만 표시, hover 시 삭제 버튼

**Step 9. FigmaContent 컴포넌트 수정**
- 파일: `smart-fnb-front/src/features/devspec/ui/FigmaContent.tsx` (수정)
- props 변경: { content, onSave } → { devSpecId }
- 상단: 입력 폼 (제목 + 설명 + URL + 저장 버튼)
- 하단: CommonCardForList 카드 그리드
- 카드 클릭 → window.open(url, "_blank")

**Step 10. DevSpecDetail 연결 수정**
- 파일: `smart-fnb-front/src/features/devspec/ui/DevSpecDetail.tsx` (수정)
- FigmaContent에 devSpecId={detail.id} 전달로 변경

## 검증
1. 백엔드: ./gradlew compileJava
2. 프론트: npx tsc --noEmit
3. Swagger UI에서 figma-links API 테스트
4. 브라우저에서 Figma 탭 → URL 입력 → 저장 → 카드 목록 표시 → 클릭 시 새탭
