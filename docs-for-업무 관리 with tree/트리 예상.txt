================================================================================
4. 인터랙션 정리
================================================================================

[트리 조작]
- 노드 클릭        -> 오른쪽에 해당 항목 상세 표시
- v / > 토글       -> 하위 항목 접기/펼치기
- 우클릭 (또는 ...)-> 컨텍스트 메뉴: 이름 변경 | 하위 추가 | 삭제 | 순서 변경
- 드래그 & 드롭    -> 순서 변경 (sortOrder 업데이트) [향후 확장]

[상세 화면 조작]
- 상태 드롭다운    -> TODO / IN_PROGRESS / DONE 변경 (즉시 저장)
- 내용 편집        -> 텍스트 에디터에서 수정 후 [저장] 버튼
- [삭제] 버튼      -> 확인 다이얼로그 후 삭제 (하위 항목도 cascade)
- [+ 하위 업무 추가]-> 새 항목 생성 모달/인라인 폼

[PROJECT 노드 전용]
- 하위 항목 진행률 요약 (프로그레스 바)
- 페이지별 현황 대시보드

[PAGE 노드 전용]
- 하위 업무 테이블 뷰
- 하위 업무 추가 버튼
