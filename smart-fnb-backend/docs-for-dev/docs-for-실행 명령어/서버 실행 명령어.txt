Gradle 기반 Java/Spring 프로젝트**입니다. 실행 명령어는:

```bash
./gradlew bootRun
```

참고로 다른 주요 명령어들도 있습니다:

| 명령어 | 설명 |
|--------|------|
| `./gradlew bootRun` | 애플리케이션 실행 |
| `./gradlew build` | 빌드 |
| `./gradlew test` | 테스트 실행 |
| `./gradlew clean build` | 클린 빌드 |
